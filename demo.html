Step 1 — The command

One command. Any AI endpoint.

aigrc is a CLI. You point it at any OpenAI-compatible endpoint — production, staging, or a mock for testing — and it runs the full OWASP LLM01 prompt injection taxonomy against your system in under a minute.

bash
$ aigrc check prompt-injection \ --target https://api.acme-fintech.com/v1/chat/completions \ --model gpt-4o-mini \ --report-json evidence.json \ --report-md audit-binder.md \ --report-sarif aigrc.sarif \ --fail-below 90

Three flags worth noting. --report-sarif emits the OASIS standard format that GitHub Code Scanning, GitLab, and Azure DevOps consume natively. --fail-below 90 turns this into a CI gate — your pipeline fails if pass rate drops below the threshold. --offline (not shown) runs against a deterministic mock for testing without burning API credits.
